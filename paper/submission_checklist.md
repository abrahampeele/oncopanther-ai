## Notes on Briefings in Bioinformatics

- **No APC for Application Notes** if corresponding author is in a non-OA institution (check current policy)
- **Open Access option** available (~£2,500 / $3,000 APC) — not required
- **Review timeline**: typically 4–8 weeks
- **No page limit** but word limit is firm at 3,000 for Application Notes
- **Accepted formats**: Word, LaTeX (template available on journal website)
- **Figure generation**: All figures must be reproducible from the code (include scripts)
